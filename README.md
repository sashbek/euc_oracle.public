# euc_oracle
Oracle system for pwm, speed and over parameter control for leaperkim (veteran) EUCs.

## About Third Eye Authority
Third Eye Pro is a commertial device and its developer restricted sharing the code of the module which drives its BLE protocol. I highly recomend you to buy original oracle wheel_module to use it for your extentions. You also can ask me to change the firmware of this module for you is you need to use it with Third Eye Pro device.
